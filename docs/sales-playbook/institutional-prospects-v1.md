## 🔧 Données à enrichir (scout IA tâche suivante)

Pour chaque prospect, le scout institutional doit compléter :
- `email_direct` : email nominatif (via LinkedIn Sales Nav + pattern guessing)
- `linkedin_url`
- `twitter_handle` (facultatif mais utile pour icebreaker)
- `latest_news` : 1-3 actualités récentes (presse, rapport publié, intervention)
- `reported_to` : hiérarchie directe (nom + fonction)
- `budget_line_approx` : ligne budgétaire 2025-2026 dont dépend l'achat
- `preferred_language` : FR / EN / AR / ES / PT
- `best_contact_time` : morning / afternoon / TZ

Table DB suggérée : `institutional_prospects` (à créer si on valide le lancement).
